# 🤖 Basic Chatbot

## 📌 Project Overview

This project is a simple rule-based chatbot built using Python. The chatbot responds to predefined user inputs with predefined replies. It demonstrates fundamental Python programming concepts such as conditional statements, functions, loops, and input/output operations.

---

## 🎯 Objective

Build a chatbot that can:

* Greet users
* Respond to simple questions
* End conversations when requested
* Demonstrate basic conversational flow

---

## ✨ Features

* Responds to user greetings such as **"hello"**
* Answers simple questions like **"how are you"**
* Ends the conversation when the user types **"bye"**
* Uses functions for code organization
* Uses loops to keep the conversation running
* Uses if-elif-else statements for decision making

---

## 🚀 How It Works

The chatbot continuously asks for user input and checks the input against predefined rules.

### Example Conversation

You: hello

Bot: Hi!

You: how are you

Bot: I'm fine, thanks!

You: bye

Bot: Goodbye!
